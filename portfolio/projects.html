To-Do App

A simple web app that allows users to manage tasks, with persistent storage via localStorage.

This project helped me understand how to work with the browser's local storage API. I implemented features like task creation, deletion, and real-time persistence. It was relatively simple at first but required careful planning to handle dynamic DOM updates. The most complex part was ensuring data consistency after multiple user actions without a backend server.
